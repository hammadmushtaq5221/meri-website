# My Website

A responsive website built with HTML, CSS, and JavaScript, deployed on Azure Web App.

## Features
- Fully responsive design
- Mobile-friendly navigation
- Contact form with validation
- Smooth scrolling
- Deployed on Microsoft Azure

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- Azure Web Apps
- GitHub for version control

## Deployment on Azure
This website is deployed on Azure Web App service. Follow these steps to deploy your own:

1. Create an Azure account
2. Create a Web App resource
3. Connect your GitHub repository
4. Configure deployment settings
5. Deploy!

## Local Development
To run this website locally, simply open `index.html` in your browser.
